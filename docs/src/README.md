# Tài liệu theo cấu trúc `src`

Thư mục này mirror cấu trúc `src` để tra cứu nhiệm vụ theo module.

Mỗi file `.md` tương ứng với một file nguồn chính và mô tả:

- Trách nhiệm của module.
- Dữ liệu vào/ra quan trọng.
- Luồng nghiệp vụ liên quan.
- Điểm cần cập nhật khi áp dụng role mới `host`, `teacher`, `student`, `viewer`.

Các luồng tổng quan nằm ở:

- `docs/LUONG_DANG_NHAP_DE_XUAT.md`
- `docs/PHAN_QUYEN_DE_XUAT.md`

# `src/ui/styles/theme.css`

## Vai trò

Định nghĩa toàn bộ style cho app: layout, dashboard, cards, form, modal, notification, calendar và responsive.

## Task cần cập nhật

- Khi thêm màn onboarding/chờ duyệt/duyệt hồ sơ, tái sử dụng style panel/form hiện có.
- Khi thêm màn host theo giáo viên, dùng layout dashboard hiện tại nhưng tránh lồng card quá sâu.
- Kiểm tra responsive sau khi thêm tab hoặc panel mới.

# `src/ui/components/FilterBar.js`

## Vai trò

Tạo thanh lọc danh sách học viên.

## Trách nhiệm chính

- Render các control filter.
- Gọi handler khi thay đổi filter.
- Hỗ trợ tìm kiếm, trạng thái lý thuyết, sa hình, học phí, DAT, loại bằng.

## Task cần cập nhật

- Các filter liên quan học phí hoặc dữ liệu nhạy cảm nên ẩn nếu role không có quyền xem.
- Filter chỉ lọc dữ liệu đã được phép đọc.
